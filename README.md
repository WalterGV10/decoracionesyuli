# Decoraciones Yuli ✨

Plataforma premium de diseño de eventos especializada en decoración con globos, floristería y planificación de momentos inolvidables.

![Decoraciones Yuli](public/decoyulis.png)

## 🚀 Tecnologías
Este proyecto está construido con lo último en desarrollo web moderno:
- **Next.js 15+** (App Router)
- **TypeScript** para máxima seguridad tipográfica
- **Tailwind CSS** con diseño minimalista y *Glassmorphism*
- **Framer Motion** & **GSAP** para animaciones fluidas y efectos de física
- **Lucide React** para iconografía moderna

## 🪄 Características Principales
- **Galería Interactiva:** Efecto "Bounce Cards" elástico para destacar trabajos principales.
- **Secciones Modulares:** Arquitectura limpia con componentes independientes para Bodas, XV Años y Servicios.
- **Diseño Responsivo:** Optimizado para dispositivos móviles con menú hamburguesa y navegación fluida.
- **Contacto Directo:** Integración con WhatsApp, Instagram, TikTok y más.

## 🛠️ Instalación y Desarrollo

1. Clona el repositorio:
   ```bash
   git clone https://github.com/WalterGV10/decoracionesyuli.git
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
